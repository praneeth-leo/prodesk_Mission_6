# ShopZone

ShopZone is a React and Vite shopping application built with product browsing,
product details, cart management, and a contact page.

## Features

- Home page with a premium ShopZone landing section
- Product listing page using API data
- Product details page with add-to-cart functionality
- Cart page with quantity and total price calculation
- Contact page
- Responsive luxury UI using green, gold, and black colors
- React Router for page navigation
- Context API for cart state management

## Tech Stack

- React
- Vite
- React Router DOM
- CSS

## How To Run

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```

## Project Note

I created the basic structure of this React shopping app first, including the
pages, routing, product cards, product details page, cart page, contact page,
and cart context. After that, I used AI assistance to debug errors, improve the
code structure, fix import issues, complete missing components, and enhance the
UI into a premium green, gold, and black luxury theme.
